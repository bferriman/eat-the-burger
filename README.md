# eat-the-burger

## Description

A silly app built to practice MVC implementation.  The user can create burgers, which are added to a database.  The user can "eat" the burger, moving it to the eaten column.

## Technologies Used

Implements MVC design pattern

MySQL

Node

Express

Handlebars

ORM (custom)

Bootstrap
